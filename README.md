# Employee Enrollment System

This project is a web-based Employee Enrollment System developed using Spring Boot. It allows administrators to manage employee data and provides authentication using JWT.

## Features

* Employee Registration
* Login Authentication using JWT
* View Employee List
* Update Employee Status
* Role-based access (Admin / Employee)

## Technologies Used

* Java (Spring Boot)
* MySQL Database
* Spring Security + JWT
* HTML, CSS, JavaScript

## Project Structure

* Controller Layer: Handles API requests
* Service Layer: Business logic
* Repository Layer: Database interaction
* Security Layer: Authentication and authorization

## Setup Instructions

1. Import project into IDE
2. Configure MySQL in application.properties
3. Run SQL script from db folder
4. Start Spring Boot application
5. Open browser at:
   http://localhost:8080
